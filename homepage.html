About the Product

We are building an AI based personalised learning platform that provides content and testing based on individual characteristics, Our platform collects hundreds of datapoints for each student showing them their weaknesses, strengths and builds custom curriculum for each student, this allows each student to learn on his/her own pace.

    Exams we plan to focus on:
  • CAT (live)
  • GRE (Coming soon)
  • GMAT (Coming soon)

    Feature list

  • Customized practice sessions for each student : wherein each student will be given practice problems based on his/her weaknesses, strengths and these will keep changing as the student performance increases.
  • Full mock test for CAT : this will show his/her expected percentile performance, detailed analysis and actionable insights to perform better next time.
  • Progress Tracker: this will show each individual's performance on the practice sessions overtime, performance will include accuracies on various topics, average time taken, expected percentile/rank.
  • Guidance from experts : CAT experts will offer personalised advice based on the data collected during the practice session and full mock test.
